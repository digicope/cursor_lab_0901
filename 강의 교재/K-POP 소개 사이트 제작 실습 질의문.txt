[K-POP 소개 사이트 제작 실습 질의문]

프로젝트 폴더명 : K-POP_Site_Project

1단계: HTML 기본 구조 홈페이지 작성

K-POP을 소개하는 심플한 홈페이지를 HTML만 사용하여 만들어줘.
제목, 간단한 소개 문구, 대표 K-POP 그룹 리스트(예: BTS, BLACKPINK, EXO)를 넣어줘.

2단계: 디자인 개선 (내부 CSS 활용)

디자인이 너무 단순하네. 파란색 톤을 메인 색상으로 해서 좀 더 멋지게 만들어줘.
제목은 크게, 그룹 리스트는 카드 형태로 보여줘. CSS는 <style> 태그로 HTML 내부에 포함시켜줘.

3단계: 외부 CSS 분리

지금까지 작성한 디자인을 외부 CSS 파일로 분리해줘.
HTML 파일과 style.css 파일을 따로 작성해주고, HTML에서 CSS 파일을 불러오도록 연결해줘.

4단계: 이미지와 미디어 추가

각 K-POP 그룹 옆에 대표 이미지를 넣어줘.
또한, 유튜브에서 K-POP 뮤직비디오 하나를 iframe으로 삽입해줘.

//그룹대표 이미지 출력링크는 수동으로 개별 수정 필요


5단계: 네비게이션 메뉴 추가

홈페이지 상단에 네비게이션 메뉴를 추가해줘.
예: '홈 | 그룹 소개 | 갤러리 | 연락하기' 형태의 메뉴를 만들어서 페이지 상단에 고정되도록 해줘.

6단계: 반응형 디자인 적용

모바일에서도 잘 보이도록 반응형 디자인을 적용해줘.
그룹 리스트가 PC에서는 가로 3열, 태블릿에서는 2열, 모바일에서는 1열로 보이게 해줘.

7단계: 간단한 JavaScript 추가

JavaScript를 이용해서 버튼 클릭 시 'K-POP을 사랑해주셔서 감사합니다!' 라는 알림창(alert)이 뜨도록 기능을 추가해줘.

8단계: 연락하기 폼 추가

연락하기(Contact) 섹션을 만들어줘.
이름, 이메일, 메시지 입력폼과 전송 버튼을 만들어주고, 간단한 입력 검증(이메일 형식 확인)도 해줘.


(옵션) :추가 기능 구현
K-POP 관련 최신 뉴스를 검색하고 그 결과 내용을 반영해서 “최근 K-POP 소식” 섹션에
넣어줘.
