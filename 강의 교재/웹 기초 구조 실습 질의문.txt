[웹 기초 구조 실습 질의문]

프로젝트 폴더명 : Web_Basic_Project

1단계: HTML 기본 구조

웹 페이지의 최소한의 HTML 기본 구조를 작성해줘.
- <!DOCTYPE html>, <html>, <head>, <body> 포함
- <h1> 제목, <p> 단락, <a> 링크, <img> 이미지 예시 추가

2단계: CSS 기본 적용

위 HTML 코드에 CSS를 추가해줘.
- 내부 <style> 태그 사용
- 배경색 변경, 제목 색상 변경, 단락 글꼴 크기 조정
- 링크에 hover 시 색상 변화 추가

3단계: 외부 CSS 분리

CSS 코드를 별도 style.css 파일로 분리해줘.
- HTML에서 <link rel="stylesheet">로 연결
- 동일한 스타일이 적용되도록 작성

4단계: JavaScript 기초 추가

위 HTML에 JavaScript를 추가해줘.
- <button> 클릭 시 alert("버튼이 클릭되었습니다!") 실행
- <script> 태그를 HTML body 하단에 배치

5단계: DOM 조작 실습

HTML 요소를 JavaScript로 동적으로 변경하는 예제를 추가해줘.
- 버튼 클릭 시 <p> 단락의 텍스트를 "안녕하세요, JavaScript!"로 변경

<여기까지만 실습>
-------------------------------------------------------------------------------------------
<추가 실습: 여기부터는 옵션>

6단계: 이벤트 처리

JavaScript로 여러 이벤트를 처리하는 예제를 추가해줘.
- 버튼 클릭 시 텍스트 변경
- 마우스를 이미지에 올리면 테두리 색상 변경
- 입력창에 글자를 입력하면 화면에 실시간 표시

7단계: 간단한 레이아웃 만들기
HTML과 CSS로 간단한 웹 레이아웃을 만들어줘.
- header, nav, main, footer 영역 구분
- nav에는 메뉴 3개 (홈, 소개, 연락처)
- main에는 기사 영역(제목, 내용)
- footer에는 저작권 표시

8단계: CSS Flexbox/Grid 활용

위 레이아웃을 CSS Flexbox 또는 Grid를 이용해 정렬해줘.
- nav 메뉴는 가로로 배치
- main 영역은 기사 2개가 나란히 표시되도록 구성

9단계: 간단한 동적 기능

JavaScript로 간단한 동적 기능을 추가해줘.
- nav 메뉴 클릭 시 alert("메뉴 클릭됨: 메뉴이름")
- Dark Mode 버튼 추가 (배경/글자색 전환)

10단계: 통합 미니 프로젝트

HTML/CSS/JavaScript를 활용하여 간단한 "To-Do 리스트" 웹앱을 만들어줘.
- 입력창에 할 일을 입력하고 추가 버튼 클릭 시 리스트에 항목 추가
- 각 항목 옆에 "삭제" 버튼을 눌러 항목 제거
- CSS로 리스트를 깔끔하게 꾸며줘
