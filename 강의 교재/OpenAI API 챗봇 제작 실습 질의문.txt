[OpenAI API 챗봇 제작 실습 질의문]

프로젝트 폴더명 : Chatbot_Project

1단계: 기본 환경 준비

chatbot.py를 만들어서 OpenAI API를 사용해  1회 질문을 보내고 답을 출력하는 코드를  
함수나 환경변수 사용하지 않는 최소 코드만 작성해줘.  

<터미널에서 모듈설치>
pip install openai python-dotenv

<구버전으로 실행 오류 발생시>
OpenAI  최신 버전 스타일로 다시 만들어줘. 


2단계:  GPT Model 변경

GPT 최신 모델로 변경해줘.


3단계: 환경 변수 등록 관리

현재 프로젝트 경로 바로 안에다가 .env 파일을 하나 만들고 이 파일에 소스에 들어 있는 API 키를
환경 변수로 저장해주고 API 키를 .env 파일에서 환경변수로 읽어오도록 소스 수정해줘.


4단계: 사용자 인터페이스 개선

질문 입력 시 'You:' , 응답 출력 시 'Bot:' 이 앞에 붙도록 수정해줘.

5단계: 함수 구조화 및 루프 추가

사용자 입력을 받아 OpenAI API에 전달하고, 응답을 리턴하는 함수(chat_with_gpt)를 만들어줘.
사용자가 'quit'을 입력하기 전까지 반복적으로 실행하도록 구현해줘.

6단계: 간단한 메모리 기능

대화 히스토리를 리스트에 저장해서, 이전 질문과 답변을 맥락으로 포함해주는 챗봇으로 수정해줘.

7단계: 에러 처리

API 호출이 실패했을 때 예외 처리(try-except)를 추가하고, 오류 메시지를 출력하도록 해줘.

8단계: 파라미터 선택기능

temperature 값을 0.2와 1.0 두 가지 모드로 선택할 수 있게 수정해줘.

9단계: GUI 버전 구현
지금까지 만든 예제를 Tkinter GUI로 새로운 파일로 작성해줘.


10단계:  GUI 화면 디자인 개선

GUI 화면과 대화 텍스트 폰트 및 색상을 좀더 예쁘게 만들어줘.


11단계:  대화 내용 저장 및 불러오기  구현
대화 내용을 현재 시간 정보와 함께 DB로 항상 저장하고
[History] 버튼을 누르면  대화 내용 전체를 목록으로 볼수 있는 기능을 추가해줘

<오류 발생시>
대화 내용이 안보여 다시 수정해줘.
