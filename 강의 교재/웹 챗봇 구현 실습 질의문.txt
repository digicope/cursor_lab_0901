[웹 챗봇 제작 실습 질의문]

프로젝트 폴더명 : Web_Chatbot_Project

<flask 파이썬 모듈 설치>
pip install flask

- Web Chatbot  만들기 질의문

Flask를 사용해서 간단한 웹 챗봇 서버를 만들어줘.
조건은 다음과 같아 :
1. 현재 경로에서 채팅 UI(입력창, 버튼, 채팅창 영역)를 보여준다.
2. OpenAI API 최신 버전 스타일 코드로 만들어주고 모델도 "gpt-4o-mini'를 사용하도록 해줘. 
3. API 키를 현재 경로의 .env 파일에서 환경변수로 읽어와서 사용하도록 구현해줘
4. 사용자가 메시지를 입력하면 Flask 서버가 OpenAI API에 전달하고, 모델의 응답을 받아 채팅창에 표시한다.
5. 질문 입력 시 'You:' , 응답 출력 시 'Bot:' 이 앞에 붙도록 해줘.
6. 비동기(Fetch API) 요청을 사용해서 새로고침 없이 채팅이 가능해야 한다.
7. CSS로 디자인을 개선해서 사용자 메시지는 오른쪽 파란색 말풍선, 챗봇 메시지는 왼쪽 회색 말풍선으로 보여준다.
8. 실행에 필요한 requirements.txt 파일도 함께 만들어줘."


- 기능 개선 (1)

대화내용이 많아지면 하단 입력 창이 보이질 않게 되니  하단 입력창은 고정시키고 메시지 화면을 스크롤 가능하게 수정해줘.

- 기능 개선 (2)

카카오톡 채팅방과 똑같은 디자인으로 만들어줘.
